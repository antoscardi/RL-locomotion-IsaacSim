# RL-locomotion-IsaacSim
ROS 2 Humble node for real-time RL-based locomotion control of the Unitree G1 humanoid robot. Subscribes to odometry, joint states, velocity commands, and elevation maps to build an observation tensor and run inference on a trained policy network (IsaacLab/IsaacSim). Includes pre-trained walking model with welder payload.
